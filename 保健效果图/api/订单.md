```$xslt
type_id:类别;901普通;902处方
goods_ids:商品ID1,商品ID2
address_id:地址ID
customer:王坤                          收货人
id_card:110102199901014034            身份证号

```

# 用户自己上传 处方药
url:/index/order/img_user

post:
```html
file:           上传文件字段名称
```
注意：这里是上传图片，`enctype`属性必须是`multipart/form-data`

return:
```html
ret: 状态  1 成功  0失败
msg: 说明
img_user: 图片地址
```
