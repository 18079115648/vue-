# 用户头像
url:/index/user/img_head

post:
```html
file:           上传文件字段名称
```
注意：这里是上传图片，`enctype`属性必须是`multipart/form-data`

return:
```html
ret: 状态  1 成功  0失败
msg: 说明
img_head: 头像地址
```
