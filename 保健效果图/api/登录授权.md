# 微信 授权
url：/index/api/weixin

get:
```html
url:  回调链接  urlencode
```
RETURN:
```html
如果没有授权，此链接直接跳转到微信进行授权，授权完成后，会跳转到指定  url 地址里。
如果已授权，此链接会 对  当前  微信用户尝试登陆，

如果用户不存在， 直接跳转到/login 进行手机绑定(手机绑定成功后，返回 制定的url,具体看绑定接口)
```
