# 健康 创建

url:/index/health/create

post:
```SHELL
    true_name:                       姓名
    birthday：1999-01-01             生日 
    height：180                      身高厘米 
    sex：                            性别 1 男 2女 0保密
    weight：75000                    体重 克   
    relationship_id：                关系ID      当 is_my 为1时，这里默认0
    is_my：                          是否自己  1是  0 否   
    data:处方药,健康标签               健康标签，多个用,号隔开
```
return:
```SHELL
ret: 状态  1 成功  0失败
msg: 说明
health_id                           健康ID
```
