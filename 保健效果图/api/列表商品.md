# 评论创建 商品

url:/index/goods/commentCreate

post:
```
order_id：订单ID
goods_ids:[商品id 1,商品id 2]
score:[评分1,评分2]
content_json:["内容1","内容1"]                      json 格式
score_service:服务评分
score_speed:送货速度评分
```

RETURN:
```angular2html
ret: 状态  1 成功  0失败
msg: 说明
```
