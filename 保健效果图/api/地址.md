# 地址 更新

url:/index/address/update

post:
```html
    id:                                                     主键ID 地址ID

    contact:                                                姓名

    mobile:                                                 手机号

    id_card:                                                身份证

    province:                                               省ID

    city:                                                    市ID

    district:                                               区ID

    address:                                                地址
    
    is_default:                                    是否默认地址；0否，1是
```
RETURN:
```angular2html
ret: 状态  1 成功  0失败
msg: 说明
```
