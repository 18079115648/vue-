# 搜索历史

url:/index/search/history

get:
```angular2html
size:每页条数   默认 10条
```
return:
```angular2html
ret: 状态  1 成功  0失败

msg: 说明

data：内容
    [
    [
        "wd":"搜索的关键词"
    ]
    ]
```
